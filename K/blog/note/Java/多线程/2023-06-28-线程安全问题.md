**线程安全**：当多个线程并发访问某个Java对象（Object）时，无论系统如何调度这些线程，也无论这些线程将如何交替操作，这个对象都能表现出一致的、正确的行为，那么对这个对象的操作是线程安全的。

## 一、自增运算不是线程安全的

自增运算符是一个复合操作，至少包括三个 JVM 指令：“内存取值”、“寄存器增加1”和“存值到内存”。这三个指令在 JVM 内部是独立进行的，中间完全可能会出现多个线程并发进行。
