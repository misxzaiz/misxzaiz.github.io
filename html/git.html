常用命令

初始化仓库

git init [文件名]

添加文件

git add 文件名

提交版本

git commit -m "注释信息" 如果我们修改了很多文件，而不想每一个都add，想commit自动来提交本地修改，我们可以使用 -a 标识。 git commit -a -m "注释信息"

发布版本

git clone ssh git push ssh

取回更新

如果您已经按上面的进行push，下面命令表示，当前分支自动与唯一一个追踪分支进行合并。 git pull 从非默认位置更新到指定的url。 git pull url

删除

git rm 文件名

分支与合并

主分支（master） 创建分支 git branch 分支名 更改当前分支 git checkout 分支名 合并分支（切换主分支） git checkout master git merge 分支名 删除分支 git branch -d 分支名 :::
